
🪙 Crypto App - Cryptocurrency Tracking App
Crypto App is a web application developed using React and Vite that allows the user to:

View up-to-date cryptocurrency rates

Add their crypto assets to their portfolio

Monitor price movements

Manage their investments in real time

⚙️ Technologies:
React + Vite - fast and modern frontend

JavaScript (ES6+)

Node.js + Express (optional if server side)

Styled Components - interface styling
